# Page Sections

Large, page-specific sections that compose the main content areas.

## Examples

- `Hero.tsx` - Landing page hero with CTA
- `Projects.tsx` - Portfolio projects showcase
- `Skills.tsx` - Tech stack and skills display
- `Contact.tsx` - Contact form section

## Guidelines

- **Page-specific** - Tied to specific pages/routes
- **Composable** - Built from smaller UI components
- **Server Components** - Default to Server Components
- **Performance** - Lazy load images, optimize bundle
- **Responsive** - Mobile-first, works on all screens
