# Shared Layout Components

Persistent layout components that appear across multiple pages.

## Examples

- `Header.tsx` - Navigation header with logo and menu
- `Footer.tsx` - Site footer with links and copyright
- `Navigation.tsx` - Main navigation menu
- `Sidebar.tsx` - Sidebar for multi-page layouts

## Guidelines

- **Persistent** - Appear on multiple pages
- **Layout-focused** - Structure, not content
- **Server Components** - Static by default
- **Consistent** - Same look/feel across pages
- **Accessible** - Keyboard navigation, ARIA labels
