# UI Components

Reusable, generic UI components that can be used across the application.

## Examples

- `Button.tsx` - Button with variants (primary, secondary, ghost)
- `Card.tsx` - Content card with shadow and hover effects
- `Input.tsx` - Form input with validation states
- `Badge.tsx` - Small status indicators

## Guidelines

- **Reusable** - Should work in any context
- **Props-driven** - Behavior controlled via props
- **TypeScript** - Strict type definitions
- **Accessible** - WCAG 2.1 AA compliant
- **Testable** - Unit tests for variants
